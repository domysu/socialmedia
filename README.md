# Social Media Website

## Project Overview

This project is a social media website currently under development. It serves as a learning experience in using PHP, Laravel, Linux, Tailwind.

## Technologies Used

- **PHP**
- **Laravel**
- **Tailwind CSS**
- **Docker**
- **Ubuntu**
